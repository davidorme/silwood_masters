Marking assignments

The table below shows all the assignments created on the system along with their current status. Authorised users can use the view links to look at and edit reports: this ability is needed to fix problems or moderate reports. Each assignment has one of the following status indicators.

{{=status_dict['Created']}} Created. When new assignments are created or uploaded, they are given this status. This gives coordinators a chance to check assignments before they are sent to markers and supervisors.
{{=status_dict['Not started']}} Not started. These assignments have been distributed to their marker but no marking data has yet been entered.
{{=status_dict['Started']}} Started. The marker has entered some marking data and saved their progress but the report has not yet been submitted.
{{=status_dict['Submitted']}} Submitted. The marker has completed their report and submitted it.
{{=status_dict['Released']}} Released. These reports have been released to students.

Actions

There are five actions that can be applied to sets of records. The search box above the table can be used to select subsets of records to run actions on.

Send to Markers This action will automatically distribute the marking links to markers.
Download Confidential PDFs This creates a zipfile containing the confidential versions of the report PDFs. Note that only completed reports will be included.
Download Public PDFs This creates a zipfile containing the public versions of the report PDFs that are released to students. Again, only completed reports will be included.
Download Grades This action downloads an Excel file containing the grades exported from each assignment form.
Release to Students This action emails the students with links to their reports. They will receive an email containing a set of links that will allow them to download a PDF of the report, excluding any confidential report fields. The system will only release submitted and previously released reports, so you can resend links but not accidentally release partial reports.
