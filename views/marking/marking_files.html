Marking files

The files for marking are stored on Sharepoint to make it easy to provide controlled access for markers. The folder structure and filenames in that coursework directory are then used to match files to individual student coursework preseentations.

This page shows the matching of files to presentations. The first table shows the number of presentations bu course in this academic year. The following tables then:

  • presentations that do not yet have a file and
  • files that do not match a presentation.

The button below can be used to rescan the Sharepoint folder after files have been added or corrections made. It does take some time to run and will then reload this page with updated results: be patient!
