Load marking assignments

Use the form below to upload a CSV file of marking assignments. The format of the file is the same as the downloads from the Project Allocations page. That format includes the lead supervisor but you will need to add columns to that file containing the emails for other markers.

The tricky bit is the mapping from markers to marking roles. Each marker may need to complete several assignments (e.g. marking a thesis and a presentation): each assignment needs a role name and a due date. The default mapping below is standard for Silwood, but can be altered. It uses the YAML format. The 'marker names' (e.g. 'marker_1') match the emails in the file to each set of roles.
