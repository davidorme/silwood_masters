My Projects

This table shows all of the projects for which you are the lead supervisor. This includes both filled projects - when the student you supervised and the year will be shown - and also open project proposals. Students can see all project proposals that were created within 60 days of the start of this academic year ().

From this page, you have a number of options:

  • You can create a {{=A('completely new proposal ', _href=URL('projects', 'project_details', scheme=True, host=True))}}.
  • You can edit the details of a proposal ( ), including to accept a student on to a project.
  • You can refresh an open proposal ( ). All this does is to take a older proposal that students cannot see and update its creation date to make it visible.
  • You can clone any proposal ( ). This will duplicate the proposal and set today as the creation date, so it is visible. If you clone a filled project, the clone will be reset to be an open project.
