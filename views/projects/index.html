Silwood and NHM Masters Project proposals

This is a live list of projects proposals put forward by staff at Silwood and the NHM as well as staff elsewhere at Imperial College London and at other external project partners. Supervisors will continue to add projects to this list throughout the first few months of the courses, so do keep checking back!

Project details

The info links () will take you to the details of a particular project. You can also use the search box : simply enter some text and press Search to do a free text search. If you want to search more finely, the pop up search options allow you to search on particular details of the project descriptions.

External projects

While many project proposals are lead by internal supervisors at Imperial College London or the Natural History Museum ( ), some projects are lead by supervisors at external institutions ( ). If you are interested in one of these projects, please read this extra information about external projects.

Open and filled projects

The list shows both open projects ( ) and filled projects ( ), where a supervisor has already accepted a student onto a project. If you are particularly interested in the topic, it is worth contacting the supervisor to see if they can offer other similar projects but you should not expect this to be possible.
