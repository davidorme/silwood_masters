{{pass}}
  • The page slug should be a simple label that is used as a URL to point to the page. It should only contain lowercase letters and hyphens: this-is-a-valid-slug but This_One_is_Not!.
  • The page wikicontent contains the content of the wiki page, written using Markdown and including the extensions described here.
  • The Toc Slug is a link to a separate page that is used to populate the table of contents. This page will typically be an unordered list, possibly nested, containing links to other pages. You will need to edit this page to add new pages into the navigation.
